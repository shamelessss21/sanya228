# Курсовая работа: планировщик задач на основе DAG (C++)

Проект реализует практическую часть курсовой: управление выполнением задач с зависимостями через ориентированный ациклический граф (DAG), топологическую сортировку и многопоточность.

## Ключевые возможности

- построение DAG из задач и зависимостей;
- проверка графа на циклы;
- топологическая сортировка (алгоритм Кана);
- многопоточное выполнение задач с учетом зависимостей;
- демонстрационные сценарии и проверка обнаружения цикла;
- автоматические тесты на корректность алгоритмов и планировщика.

## Структура

- `include/dag_scheduler.hpp` — модель задачи, граф и планировщик;
- `src/main.cpp` — демонстрация и тестовые сценарии;
- `tests/test_scheduler.cpp` — автотесты;
- `docs/coursework_notes.md` — пояснения для текста курсовой;
- `CMakeLists.txt` — сборка проекта и тесты.

## Сборка и запуск

### Вариант 1 (через CMake)

```bash
cmake -S . -B build
cmake --build build
./build/dag_scheduler
ctest --test-dir build --output-on-failure
```

### Вариант 2 (одной командой через g++)

```bash
g++ -std=c++20 -O2 -pthread -Iinclude src/main.cpp -o dag_scheduler
./dag_scheduler
```

## Что заменено относительно предыдущей версии

Предыдущая реализация в репозитории была на Java. По требованию практическая часть возвращена на C++ с полной переработкой архитектуры, сборки и тестирования.
